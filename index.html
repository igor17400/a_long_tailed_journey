Methods for Long-Tail Learning

We study two complementary families of techniques: re-balancing strategies that modify the loss function to counteract class imbalance, and representation learning approaches that use contrastive objectives to learn embeddings where same-class samples cluster together.

Re-balancing LDAM

Label-Distribution-Aware Margin Loss. Enforces class-dependent margins proportional to class frequency, pushing the decision boundary away from rare classes.

Δy = C / ny1/4

Re-balancing Logit Adjustment

Shifts logits by class priors to achieve Fisher-consistent balanced error. Simple post-hoc or during-training correction.

logity → logity + τ · log(πy)

Routing RIDE

Routing Diverse Distribution-Aware Experts. Multiple expert branches with LDAM loss and KL-divergence diversity, so each expert specialises in different regions of the class distribution.

Contrastive SupProto

Fixed antipodal class prototypes on the unit sphere. Minority and majority prototypes are placed at opposite poles, with a conditional pull loss that prevents collapse.

pmin = −pmaj

Contrastive SupMin

Class-routed contrastive loss: applies SupCon only within the minority class and NT-Xent for the majority, preventing the minority from being drowned out.

Fusion FuseMoE

Mixture-of-Experts Transformer for multimodal fusion. Laplace-gated TopK routing across modalities (vitals, labs, clinical notes), adapted for the 1% prevalence setting.

Clinical Task

Pressure Ulcer Prediction in ICU

  • Task: predict hospital-acquired PU onset using cumulative observation windows generated every 3h from admission (with 1h lead time to prevent label leakage).
  • Data: MIMIC-IV multimodal ICU data — irregular vitals, labs, and clinical notes.
  • Training: cadenced by positives — the model validates every 1,458 positives (25%) and stops after 5,831 positives (100%), preventing the majority class from dominating gradient updates.

Preliminary Results

  • Controlled testbed: CIFAR-10-LT with binary prevalence set to 1% (matching clinical PU rates) to isolate the effect of imbalance from clinical complexity.
  • Imbalanced test set: unlike standard benchmarks that evaluate on a balanced test split, we keep the test set imbalanced to reflect real-world deployment.

Conclusions

  • Testbed vs. clinical transfer: SupProto and SupMin outperform BCL and SupCon on binary CIFAR-10-LT, confirming their design for extreme imbalance. Yet contrastive losses paired with FuseMoE underperform CE on AUPRC, suggesting that representation-level fixes alone do not transfer to multimodal clinical data. MoE-based routing (RIDE) remains a promising direction.
  • Multimodal gap: Long-tail methods validated on unimodal benchmarks do not straightforwardly transfer to multimodal fusion, highlighting the need for strategies designed jointly with the fusion architecture.
  • Next steps: Design long-tail strategies purpose-built for multimodal clinical fusion.
